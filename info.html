Information

QRCodes:

  • QRCode data can be used to track the return of the product.
  • You can find the RMA within a QRCode to print out a barcode to be used as a legacy return.
  • Make sure you click generate to make a new QR Code, this will not auto generate a new QRCode

Barcodes:

  • Barcodes can be used to track the return of the product.
  • you can also use this tool to make new barcodes for dammaged barcode packages.

Tool Info:

This tool does not upload, share, or save QRcode & Barcode information. Everything is processed locally

Printer Setup

Make sure you have a printer settings saved before you print a QRCode or Barcode.

The following setup guide is for Zebra GX420d lable printers:

  • Search for softwear center on the desktop
  • search for zebra in the application
  • look for Zebra Drivers (Lables), then install.
  • Next, open windows settings app, then Bluetooth & other devices, then printers & scanners.
  • Add the zebra printer (ZDesigner GX420d) then click manage
  • In printer properties, In the general tab, click preferences at the bottom
  • set the size to width: 2.00 and height to 2.00, make sure to apply changes.
  • Next, in the advanced setup tab, set the operation mode to cutter.
  • lastly, when printing from the webpage, the printing settings should have margins set to none and the layout to portrait.
